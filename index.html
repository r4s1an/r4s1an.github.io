// 03 — Projects

Selected Work

Data Analysis for Emerging Materials in Hybrid Perovskite Photovoltaics

An open-source Python tool that automates the analysis and reporting of current-voltage measurements from perovskite solar cell devices. Automatically extracts key performance parameters — fill factor, power conversion efficiency — and generates standardised PDF reports with J-V curves, performance tables, and box plots.

Python NumPy Pandas Matplotlib SciPy Perovskite Solar Cells Photovoltaics

AI-Powered Technical Document Analysis Pipeline

An end-to-end pipeline for parsing and querying technical drilling reports. A fine-tuned DocLayout-YOLO model detects document structure, PaddleOCR and EasyOCR handle extraction, and all content is stored in PostgreSQL, queryable through a Streamlit chatbot powered by Gemini via a TAG (Table-Augmented Generation) pipeline.
